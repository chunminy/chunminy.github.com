Unity script function 執行順序

一般建立Unity Script時，Editor產生檔案後會讓該Script繼承MonoBehaviour。

由於Unity是一個蠻高階的引擎系統，所以需要一個基本class掛載各種介面包山包海的讓系統可以快速串接運行，而MonoBehaviour就是這個Class。

MonoBehaviour 本身提供很多接口讓開發者來繼承使用，所以使用MonoBehaviour最先也最重要的就是需要瞭解Unity系統在運行時，MonoBehaviour的接口呼叫順序，知道這個順序後才有辦法思考各種功能應該放在哪個function裡面才能正常運作。
