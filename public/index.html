Hexo 生成檔案附帶日期

使用Hexo 建立新文章時，只需要下指令hexo n [title]就會自動生成md檔
若是希望檔名可以附帶日期以便整理，可以修改_config.yml

# Writing
new_post_name: :year-:month-:day.md # File name of new posts
參數 作用
:title 標題
:year 年份
:month 月份
:day 日期

如此一來自動產生的檔案就會變成2016-05-28.md

附帶一提，若是有相同名稱的檔案則會自動補上count

例如

2016-05-28.md
2016-05-28-1.md
2016-05-28-2.md

sublime text 移除空白

為了避免下意識多按的空白鍵造成程式碼夾帶多餘的空白，翻找了一下sublime text的設定
只需要在 Preferences->Settings-User 調整參數就可以了

"trim_trailing_white_space_on_save": true,
"draw_white_space": "all",
"translate_tabs_to_spaces": true,
"tab_size": 4,

##trim_trailing_white_space_on_save
在存檔時會自動刪除每一行後面多餘的空白

##draw_white_space
會直接顯示出空白或是Tab，這樣就可以分辨到底哪邊是空白哪邊是Tab了

##translate_tabs_to_spaces
按Tab時自動轉成空白鍵

##tab_size
每次案tab代表幾個空白
